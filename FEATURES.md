# ✨ Новые возможности бота

## 🎯 Анализ контента и уникальность постов

### 1. Автоматическая проверка на дубликаты

Бот теперь анализирует все предыдущие посты и проверяет новый контент на уникальность:

- **Порог схожести:** 70% (настраивается)
- **Проверка:** Последние 50 постов
- **Алгоритм:** SequenceMatcher для вычисления схожести текстов

### 2. Умная генерация постов

При генерации нового поста бот:

1. **Анализирует историю канала** (последние 7 дней)
2. **Извлекает ключевые слова** из предыдущих постов
3. **Улучшает промпт** с инструкциями избегать повторений
4. **Делает до 3 попыток** генерации уникального контента
5. **Проверяет результат** на схожесть с предыдущими постами

### 3. Анализ канала 📊

Новая кнопка "📊 Анализ канала" показывает:

- **Всего постов** в истории
- **Количество уникальных** постов
- **Процент уникальности** контента
- **Среднюю длину** постов
- **Топ-5 частых тем** в канале
- **Рекомендации** по улучшению контента

### 4. Автопубликация с проверкой уникальности

Cron задача теперь:

1. Получает историю постов канала
2. Анализирует контент
3. Генерирует промпт с учетом анализа
4. Делает несколько попыток создать уникальный пост
5. Проверяет на дубликаты перед публикацией
6. Публикует только уникальный контент

## 🔧 Как это работает

### Алгоритм проверки уникальности

```python
# 1. Нормализация текстов
text1 = text1.lower().strip()
text2 = text2.lower().strip()

# 2. Вычисление схожести (0-1)
similarity = SequenceMatcher(None, text1, text2).ratio()

# 3. Проверка порога
if similarity >= 0.7:  # 70% схожести
    return "Дубликат"
else:
    return "Уникальный"
```

### Улучшение промпта

Базовый промпт:
```
Напиши интересный исторический факт.
```

Улучшенный промпт:
```
Напиши интересный исторический факт.

ВАЖНО: Создай УНИКАЛЬНЫЙ контент, который НЕ повторяет следующие темы:
Избегай повторения этих тем: война, революция, империя, царь, битва

Часто используемые темы в канале: история, факты, события, личности, даты
Попробуй найти новый угол или аспект этих тем.

Создай свежий, оригинальный контент с новой перспективой!
```

## 🎨 Генерация с проверкой

### Процесс генерации:

1. **Попытка 1:**
   - Генерация поста
   - Проверка на дубликаты
   - Если уникальный → Готово ✅

2. **Попытка 2 (если нужна):**
   - Улучшение промпта: "Создай СОВЕРШЕННО ДРУГОЙ пост!"
   - Генерация
   - Проверка

3. **Попытка 3 (если нужна):**
   - Еще более строгий промпт
   - Финальная генерация
   - Проверка

